# Console-Quiz
Данная программа представляет собой консольную викторину, которая задаёт пользователю вопросы с вариантами ответов. Пользователь выбирает правильный вариант ответа, после чего программа проверяет его и сообщает, правильный он или нет.

В конце викторины программа подсчитывает количество правильных ответов и выводит итоговый результат, а также процент правильных ответов. Кроме того, результат автоматически сохраняется в текстовый файл.

Программа работает в текстовом режиме (консоли) и не требует установки дополнительных библиотек.
